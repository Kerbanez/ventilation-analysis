# Анализ производителей вентиляционных установок в РФ

## Цели исследования
- Выявить топ-10 производителей по прибыльности
- Проанализировать модельные ряды и сегменты рынка  
- Оценить уровень локализации производства

## Данные
- Финансовые показатели 2022-2024 гг.
- Прогноз на 2025 год
- Производственные мощности
- Уровень локализации

##Структура проекта:
- VentMan.sqlite - SQLite база для оптипизированного хранения данных и возможного их пополнения для дальнейшего расширенного анализа
- requests.txt - список выполненых запросов для экспорта файлов в Power BI
- Ventilation_analysis_10.pbix - главный BI отчет по заданной теме
- Ventilation_analysis_10.pptx - вспомогательная (урезанная) версия отчета в .pptx формате (PowerPoint)
